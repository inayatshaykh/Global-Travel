# Globe Trotter

A travel website built with React, TypeScript, Tailwind CSS, and shadcn/ui.

## Features

- Browse travel destinations and packages
- Admin panel with authentication and package management
- Responsive design

## Getting Started

```bash
npm install
npm run dev
```
